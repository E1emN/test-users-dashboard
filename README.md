# Users Dashboard (React + Vite + TypeScript)

A modern users dashboard built with React, Vite, TypeScript, Effector and Ant Design.  
Data is fetched from https://dummyjson.com/users with server-side pagination and sorting.

---

## 🚀 Features

- Users table with server-side pagination
- Server-side sorting
- User details page
- Dark / Light theme
- Theme sync across browser tabs
- 404 page
- Effector state management
- Ant Design UI

---

## 🧰 Tech Stack

- React 18+
- Vite
- TypeScript
- Effector
- React Router DOM v6
- Ant Design
- Axios
- Node 24.14.1

---

## 📦 Installation

### Clone repository

```bash
git clone https://github.com/E1emN/test-users-dashboard
cd test-users-dashboard
npm install
npm run dev
```
